# RAG-Document-Q-A-Chatbot
Local, privacy-first RAG chatbot for querying your own documents — no data leaves your machine. Built with LangChain, ChromaDB, HuggingFace embeddings, and Ollama (Llama 3.2), with a Streamlit UI. Supports MMR, BM25+Chroma ensemble, and contextual compression retrieval, plus a custom local evaluator.
